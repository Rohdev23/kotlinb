# kotlinb
**Script para compilar arquivos Kotlin no Termux.**

Faça `chmod +x kotlinb` para dar permissões ao script.

Em seguida renomei o script para `kotlinb`. 

mv `kotlinb.sh kotlinb`

Em seguida copie ou nova para $PREFIX/bin dessa forma poderar usar de qualquer lugar.

Faça `mv kotlinb $PREFIX/bin`

**Para usar basta fazer**.

`kotlinb arquivo.kt`
